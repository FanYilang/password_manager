  🔐 密码管理器 - GitHub Pages 部署步骤
═══════════════════════════════════════════════════════════

✅ 第一步：启用 GitHub Pages
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开浏览器，访问你的 GitHub 仓库：
   https://github.com/FanYilang/password_manager

2. 点击顶部的 "Settings" 标签

3. 在左侧菜单找到 "Pages"

4. 在 "Build and deployment" 部分：
   ┌─────────────────────────────────────┐
   │ Source: Deploy from a branch        │
   │ Branch: main          / (root)  💾  │
   └─────────────────────────────────────┘

5. 点击 "Save" 按钮

6. 等待 1-2 分钟，刷新页面会看到：
   ✅ Your site is live at 
   https://fanyilang.github.io/password_manager/


✅ 第二步：访问你的网站
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 在手机上：
   1. 打开浏览器
   2. 访问：https://fanyilang.github.io/password_manager/
   3. 添加到主屏幕（像安装 App）

💻 在电脑上：
   1. 打开浏览器
   2. 访问：https://fanyilang.github.io/password_manager/
   3. 建议添加到书签


✅ 第三步：设置主密码
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

首次打开网站：
   ┌─────────────────────────────────────┐
   │  🔐 密码管理器                      │
   │                                     │
   │  首次使用，请设置主密码             │
   │  ┌───────────────────────────────┐ │
   │  │ 输入主密码                    │ │
   │  └───────────────────────────────┘ │
   │  ┌───────────────────────────────┐ │
   │  │ 确认主密码                    │ │
   │  └───────────────────────────────┘ │
   │  [    设置主密码    ]              │
   └─────────────────────────────────────┘

⚠️  重要：请牢记主密码，忘记后无法恢复！


✅ 第四步：跨设备同步数据
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在电脑上导出：
   1. 点击 [📤 导出] 按钮
   2. 输入主密码
   3. 下载备份文件（如：password-backup-2024-12-13.json）

传输文件到手机：
   • 方法1：发送到自己的邮箱
   • 方法2：使用微信"文件传输助手"
   • 方法3：使用云盘（百度网盘、OneDrive等）

在手机上导入：
   1. 点击 [📥 导入] 按钮
   2. 选择备份文件
   3. 输入主密码
   4. 选择冲突策略：
      ○ 跳过重复项（保留现有）
      ○ 替换重复项（使用导入的）
      ● 保留较新的 ← 推荐


═══════════════════════════════════════════════════════════
  📝 常用操作
═══════════════════════════════════════════════════════════

添加密码：
   [+ 添加新凭证] → 填写信息 → [保存]

查看密码：
   点击 [👁] 按钮显示/隐藏密码

复制密码：
   点击 [📋] 按钮（30秒后自动清除剪贴板）

编辑密码：
   点击 [✏️] 按钮 → 修改 → [保存]

删除密码：
   点击 [🗑️] 按钮 → [确认删除]

搜索密码：
   在搜索框输入网站名或用户名

锁定应用：
   点击右上角 [🔒] 按钮


═══════════════════════════════════════════════════════════
  ⚠️  安全提示
═══════════════════════════════════════════════════════════

✅ 使用强主密码（至少12位，包含大小写字母、数字、符号）
✅ 定期导出备份（建议每周一次）
✅ 将备份文件保存在安全的地方
✅ 使用完毕后点击锁定按钮
✅ 不要在公共电脑上使用

❌ 不要截图包含密码的界面
❌ 不要将主密码告诉任何人
❌ 不要将备份文件和主密码存放在一起
❌ 清除浏览器数据前务必先备份


═══════════════════════════════════════════════════════════
  🆘 常见问题
═══════════════════════════════════════════════════════════

Q: 为什么手机和电脑的数据不同步？
A: 数据存储在浏览器本地，需要通过"导出/导入"手动同步。

Q: 忘记主密码怎么办？
A: 很遗憾无法找回，需要重新设置（会丢失所有数据）。

Q: 数据安全吗？
A: 非常安全！使用 AES-256 军事级加密，数据完全本地化。

Q: 可以在多个浏览器使用吗？
A: 可以，但每个浏览器的数据独立，需要手动同步。

Q: 网站打不开怎么办？
A: 1) 检查 GitHub Pages 是否已启用
   2) 等待几分钟让部署完成
   3) 清除浏览器缓存后重试
