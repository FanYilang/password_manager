# 🚀 快速开始指南

## 第一步：启用 GitHub Pages

1. 打开浏览器，访问：
   ```
   https://github.com/FanYilang/password_manager
   ```

2. 点击页面顶部的 **Settings**（设置）

3. 在左侧菜单中找到并点击 **Pages**

4. 在 "Build and deployment" 部分：
   - **Source**: 选择 `Deploy from a branch`
   - **Branch**: 选择 `main` 和 `/ (root)`
   - 点击 **Save** 按钮

5. 等待 1-2 分钟，刷新页面，你会看到：
   ```
   ✅ Your site is live at https://fanyilang.github.io/password_manager/
   ```

## 第二步：访问你的密码管理器

### 在电脑上

直接在浏览器中打开：
```
https://fanyilang.github.io/password_manager/
```

### 在手机上

1. 在手机浏览器中打开上面的网址
2. 添加到主屏幕（像安装 App 一样）：
   - **iPhone**: Safari → 分享 → 添加到主屏幕
   - **Android**: Chrome → 菜单 → 添加到主屏幕

## 第三步：首次使用

1. 打开网站后，设置一个**强主密码**
   - 建议至少 12 位
   - 包含大小写字母、数字和符号
   - 例如：`MyP@ssw0rd2024!`

2. **重要**：请牢记这个主密码！
   - 主密码无法找回
   - 忘记后所有数据将无法恢复

3. 开始添加你的密码凭证

## 第四步：跨设备同步

### 方法：导出 → 传输 → 导入

**在设备 A（如电脑）上：**
1. 点击"📤 导出"按钮
2. 输入主密码
3. 下载备份文件

**传输文件：**
- 发送到自己的邮箱
- 或使用微信"文件传输助手"
- 或使用云盘

**在设备 B（如手机）上：**
1. 点击"📥 导入"按钮
2. 选择备份文件
3. 输入主密码
4. 选择"保留较新的"

## 📌 重要提示

✅ **定期备份**：建议每周导出一次备份
✅ **安全存储**：将备份文件保存在安全的地方
✅ **记住密码**：主密码是唯一的安全保障
✅ **使用完锁定**：点击 🔒 按钮锁定应用

❌ **不要**在公共电脑上使用
❌ **不要**截图包含密码的界面
❌ **不要**将主密码告诉任何人
